Site Plan: Explora Bolivia

Student Name: Thomas Sosa

Site Name

Explora Bolivia

I chose this name because it is a direct call to action that appeals to both Spanish and English speakers. It positions the website as a modern, reliable guide for discovering Bolivia's diverse regions.

Site Purpose

The purpose of "Explora Bolivia" is to provide a dynamic travel resource for tourists. The site will offer curated information on top destinations, allow users to filter locations by activity type, and enable them to save their personal travel wishlist using local storage.

Scenarios

  • "What are the best outdoor activities to do in the Uyuni Salt Flats during the rainy season?"
  • "Is there a way to save a list of the historical museums in Sucre so I don't forget them?"

Color Schema

This site uses an earthy palette reflecting Bolivia's natural landscapes:

Primary Olive (#606c38) - Accents
Dark Green (#283618) - Navigation and Main Titles
Cream (#fefae0) - Page Background
Earth Tan (#dda15e) - Interactive Elements
Terracotta (#bc6c25) - Secondary Titles/Borders

Typography

Headings: Montserrat

Applied to all h1, h2, and h3 tags to provide a bold, adventurous brand identity.

Body Text: Open Sans

Applied to all paragraphs, lists, and navigation links for maximum readability on digital screens.
