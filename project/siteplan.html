Site Plan: Explora Bolivia

Thomas Sosa: WDD131 Final Project

Site Name

Explora Bolivia

I chose this name because it serves as a direct invitation to discover the country. It is bilingual-friendly, professional, and positions the site as a comprehensive guide for both adventure and cultural tourism in the heart of South America.

Site Purpose

The purpose of "Explora Bolivia" is to provide a dynamic platform for tourists to discover iconic destinations such as the Salar de Uyuni, Lake Titicaca, and the Madidi National Park. The site will help users plan their trips by filtering locations by region and using local storage to save their "Must-Visit" list.

Color Schema

The color palette is inspired by Bolivia's diverse natural landscapes (Andes, Valleys, and Tropics):

Primary Green (#606c38) - Accents and Icons
Dark Jungle Green (#283618) - Navigation and Headers
Cream (#fefae0) - Page Background
Earth Yellow (#dda15e) - Buttons and Highlights
Terracotta (#bc6c25) - Titles and Borders

Typography

Headings: Montserrat

Used for all major titles. This font provides a modern, bold look that suggests adventure and reliability.

Body Text: Open Sans

Used for all paragraphs and descriptions. It is highly legible, ensuring a great user experience on both mobile and desktop screens.

Wireframes

Layout plan for the home page:

Mobile View Layout

A single-column layout featuring a "hamburger" navigation menu. The content flows vertically: Hero Image -> Destination Cards (1 per row) -> Footer. This ensures ease of use on touch screens.

Desktop View Layout

A multi-column grid layout. The navigation is a horizontal bar. Destination cards are displayed in a 3-column grid to maximize the use of space on wider screens.
